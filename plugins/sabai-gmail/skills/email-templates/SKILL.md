# Email Templates Skill

Library of email templates for common scenarios.

## Template Library

### Introduction Template (`intro`)

```
Subject: Introduction - [Your Name] from [Company]

Hi [Name],

I hope this email finds you well. My name is [Your Name], and I'm [your role] at [Company].

[How you found them / mutual connection / reason for reaching out]

I'm reaching out because [specific reason - what you'd like to discuss or propose].

Would you be open to a brief [call/meeting] to [discuss topic]? I'm flexible on timing and happy to work around your schedule.

Thank you for considering this. I look forward to the opportunity to connect.

Best regards,
[Your Name]
[Title]
[Contact Info]
```

### Follow-up Template (`follow-up`)

```
Subject: Following Up - [Original Topic]

Hi [Name],

I wanted to follow up on [our conversation / my previous email] from [date/timeframe].

[Brief recap of what was discussed or proposed]

I wanted to check in on [the status / your thoughts / next steps].

Please let me know if you need any additional information from my end, or if there's a better time to reconnect.

Thank you,
[Your Name]
```

### Thank You Template (`thank-you`)

```
Subject: Thank You - [Specific Reason]

Hi [Name],

Thank you so much for [specific thing they did - taking the time to meet, providing feedback, etc.].

[One specific thing you appreciated or found valuable]

[Next step or how you plan to use their input]

I really appreciate your [time/help/insight], and I look forward to [continuing to work together / staying in touch].

Best regards,
[Your Name]
```

### Meeting Request Template (`meeting-request`)

```
Subject: Meeting Request - [Topic]

Hi [Name],

I'd like to schedule a meeting to discuss [topic].

**Purpose:** [Brief description of what you want to accomplish]

**Suggested Duration:** [30 min / 1 hour]

**Proposed Times:**
- [Option 1: Day, Date, Time]
- [Option 2: Day, Date, Time]
- [Option 3: Day, Date, Time]

Please let me know which time works best for you, or suggest an alternative if none of these fit your schedule.

Thank you,
[Your Name]
```

### Status Update Template (`update`)

```
Subject: [Project Name] Status Update - [Date/Week]

Hi [Team/Name],

Here's the latest update on [project/initiative]:

**Summary:**
[One-sentence overview of current status]

**Completed This Week:**
- [Accomplishment 1]
- [Accomplishment 2]

**In Progress:**
- [Task 1] - [Expected completion]
- [Task 2] - [Expected completion]

**Blockers/Concerns:**
- [Blocker if any, or "None at this time"]

**Next Steps:**
- [Next action 1]
- [Next action 2]

Let me know if you have any questions.

[Your Name]
```

### Decline Template (`decline`)

```
Subject: Re: [Original Subject]

Hi [Name],

Thank you so much for [thinking of me / reaching out / the invitation to X].

After careful consideration, I'm unfortunately not able to [accept/participate/take this on] at this time due to [brief, honest reason - current commitments, bandwidth, etc.].

[Optional: Suggest alternative - another person, different timing, modified scope]

I really appreciate you [reaching out / thinking of me], and I hope we can [connect in the future / find another opportunity to work together].

Best regards,
[Your Name]
```

### Referral Request Template (`referral`)

```
Subject: Referral Request - [Context]

Hi [Name],

I hope you're doing well. I'm reaching out because I'm [looking for / trying to connect with] [type of person/company].

Given your [experience in / connections to] [relevant area], I was hoping you might be able to introduce me to someone who [specific need].

Specifically, I'm looking for [characteristics of the ideal referral].

If anyone comes to mind, I'd really appreciate an introduction. I'm also happy to provide more context if that would be helpful.

Thank you so much for considering this.

Best,
[Your Name]
```

### Feedback Request Template (`feedback`)

```
Subject: Request for Feedback - [Topic]

Hi [Name],

I'm working on [project/document/proposal] and would really value your feedback.

[Brief context on what it is and why their input matters]

**Specifically, I'd love your thoughts on:**
- [Question 1]
- [Question 2]
- [Any other observations]

[Link or attachment]

If you have 15-20 minutes to review, I'd really appreciate it. I'm hoping to [finalize/submit] by [date], so any feedback before [deadline] would be ideal.

Thank you in advance for your time and insights.

Best,
[Your Name]
```

## How to Use

1. **Select template** based on the email type
2. **Customize** all bracketed sections with specific details
3. **Adjust tone** for the relationship (more/less formal)
4. **Review** for completeness and clarity
5. **Offer variations** if the user wants alternatives

## Tips

- Templates are starting points - always personalize
- Remove sections that don't apply
- Add specific details to make emails feel genuine
- Match formality to your relationship with recipient
