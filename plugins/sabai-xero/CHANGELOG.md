# Changelog

All notable changes to this plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-20

### Added
- Initial release
- Integration with official Xero MCP Server (@xeroapi/xero-mcp-server)
- Invoice management (create, list, update, void)
- Contact management (customers and suppliers)
- Payment recording
- Financial reports (P&L, Balance Sheet, Trial Balance)
- Aged receivables and payables reports
- `/invoice` command for invoice operations
- `/xero-report` command for financial reports
- `/contacts` command for contact management
- Comprehensive skill documentation with workflows
