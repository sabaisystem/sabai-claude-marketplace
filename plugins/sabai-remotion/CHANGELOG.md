# Changelog

All notable changes to this plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-20

### Added
- Initial plugin structure
- MCP server with video rendering tools
- `list_templates` tool for listing available templates
- `preview_frame` tool for quick frame previews
- `render_video` tool for full video rendering
- `get_composition_info` tool for composition metadata
- Social Clip template (9:16 vertical)
- Tutorial template (16:9 horizontal)
- Data Visualization template
- Text Animation template
- MCP App UI for interactive preview
- `/video` command for quick generation
