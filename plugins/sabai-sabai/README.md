## Overview

A relaxation-focused plugin providing a break mechanism for developers and users. The name references the Thai expression "สบายสบาย" meaning "relax, take it easy." Activates a video player with chill Thai vibes when you need a break from coding or work.

## Key Features

- Simple `/sabai` command to take a break
- Video player with relaxing Thai vibes
- Quick escape from work mode

## Use Cases

- "I need a break"
- "/sabai"
- "Play some relaxing music"

## Commands

- `/sabai` - Display video player with chill Thai vibes

## Configuration

No configuration required. Works out of the box.

## Authentication

None required.

## Dependencies

- **Required**: None (self-contained MCP App)

## Limitations

- Video content is embedded (no external streaming)
