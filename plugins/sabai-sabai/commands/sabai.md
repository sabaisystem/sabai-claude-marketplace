# /sabai

Take a break and relax with some chill Thai vibes.

## When to use

Use this command when you need a moment to relax and unwind during your coding session.

## Instructions

When the user invokes this command, call the `sabai-video-player` MCP App tool to display an embedded YouTube video player with relaxing music.

Simply invoke the tool with no arguments - it will display a video player with the Sabai Sabai vibes.
