# Changelog

All notable changes to Sabai Mermaid will be documented in this file.

## [1.0.0] - 2026-02-20

### Added
- Initial release of Sabai Mermaid plugin
- Interactive Mermaid diagram viewer as MCP App
- `render_mermaid` tool for generating diagrams from code
- Support for all Mermaid diagram types (flowchart, sequence, class, state, ER, pie, etc.)
- Built-in templates for quick diagram creation
- Undo functionality with history tracking
- Live code editing in the UI
- Copy diagram code to clipboard
- Light and dark mode support
- Sabai System branding footer
