# Changelog

All notable changes to this plugin will be documented in this file.

## [1.3.0] - 2026-02-20

### Added
- Multi-participant availability checking skill
- Free/busy aggregation across multiple calendars
- Conflict detection and resolution suggestions
- Working hours consideration across timezones
- Support for mixed internal/external participants
